Metaverse Introduction

发表于 2019-03-13 | 更新于 2019-03-15 | 分类于 blockchain

I have worked for more than a year (2017.12-2019.03) in Metaverse (https://mvs.org/).
I’d like to write sth about this great company.

Metaverse is forked from libbitcoin (which is a multipurpose bitcoin library targeted towards high end use).
Metaverse has extended the transaction’s output with attachments to support ETP, ASSET, DID, CERT and MIT, etc. For more tech info, please see Transaction format and signing

During my working period, Metaverse has released three major versions:

v0.7.5
v0.8.0 (SuperNova)
v0.9.0 (MPC (Pillars))

v0.7.5 version is deployed to fix future blocktime attack.
Please see more here Metaverse Full Node Wallet 0.7.5 Release Notes

In SuperNova version, Metaverse has added many great functions, like,

New MST features (Secondary issue, Conditional Offering, Domain-name and certification)
Digital identity (Avartar)
MIT (like ERC721 Non-Fungible Tokens)

Please see more here Metaverse Full Node Wallet 0.8.0 Release Notes
and Supernova new features

In MPC version, Metaverse has added PoW + PoS bi-consensus to prevent 51% attack.
Please see more here Metaverse Full Node Wallet 0.9.0 Release Notes
and Guide to PoS and MST mining

And there is also a great function I have to mention here, which is released in version v0.8.3. since that version, Metaverse support swap tokens between ERC20 tokens and MST, and between ETH and ETP.
please see more here Cross-chain Token Swap Instruction

In conclusion, Metaverse is a pioneer of blockchain technology, and keep abreast of the latest new techniques. Wish it keep on going forward and win a good future!
