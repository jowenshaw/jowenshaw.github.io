换源后更新系统

更改下系统设置#

关闭触摸板自然滚动
修改终端 tab 页切换快捷键（PgUp/PgDn换为方向键）

安装 chrome 浏览器#

  1. 将 chrome 下载源加入到系统的源列表（添加依赖）

  2. 导入谷歌软件的公钥，用于对下载软件进行验证

  3. 对当前系统的可用更新列表进行更新（更新依赖）

  4. 安装谷歌 Chrome 浏览器（稳定版）（安装软件）

  5. 启动谷歌 Chrome 浏览器

然后将 chrome 添加到 Dock 收藏夹即可。

更新语言包，安装 fcitx 输入法#

  1. 更新语言包：
    设置->区域和语言->管理已安装的语言

  2. 安装 fcitx 输入法：

设置->区域和语言->管理已安装的语言->键盘输入法系统设为fcitx
需要注销登录或者重启系统生效, 再对 fcitx 配置一下，删掉一些不用的输入法条目。

配置 vim, bash, 和常用脚本#

主要内容有 .vimrc, .bashrc, bash_completion.d 等。
